## Getting Started

1.  **Clone the repository:**

    ```bash
    git clone https://github.com/yasser8111/abdullahProtfolio.git
    cd abdullahProtfolio
    ```

2.  **Install dependencies:**

    ```bash
    npm install
    # or
    yarn install
    ```

3.  **Run the development server:**

    ```bash
    npm run dev
    ```

4.  **Build for production:**
    ```bash
    npm run build
    ```

## Customization

- **Colors**: Edit CSS variables in `src/index.css` (e.g., `--color-primary`, `--color-accent`) to change the site's theme instantly.
- **Content**: Update components in `src/components/` to add your own bio, projects, and contact info.
